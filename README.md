# 极速枪战 Arena

在线 **1v1 第一人称 3D 枪战对决**，手机浏览器打开即可游玩。Three.js 渲染 + Socket.IO 实时联机，**关键战斗数据由服务端权威裁决**（防前端作弊）。

## 玩法

- **单人训练**：首页选难度（简单 / 普通 / 困难）→「开始单人对战 AI」立即开打。
- **联机对战**：创建房间 → 把 4 位房间号发给好友 → 好友输入房间号加入。
- 满 2 人（或单人 + AI）自动 3 秒倒计时开战，进入对称竞技场。
- 击败对手 +1 分，被击败后 3 秒复活（1 秒无敌保护）。
- 单局 **3 分钟**，比分高者获胜；**平局进入 30 秒加时赛**，先击败对方者获胜。
- 局后显示**结算数据页**：命中率、击败数、总伤害、换弹次数、补给拾取，双方对比。

### 游戏模式

| 模式 | 说明 |
|---|---|
| 传统对战 | 经典 1v1，无补给物品 |
| 战术补给 | 地图上刷新血包、护甲板、弹药箱、疾跑芯片；护甲先吸收伤害，加速提升移动速度 |

### 武器系统

三把武器可在局中切换（`1` / `2` / `3` 键，或 HUD 武器槽），服务端权威裁决弹药/伤害/换弹/切枪：

| 武器 | 代号 | 弹匣 | 单发伤害 | 换弹 | 特点 |
|---|---|---|---|---|---|
| 突击步枪 | AR-30 | 30 发 | 20 | 1.5 s | 射速快，通用 |
| 手枪 | PX-7 | 12 发 | 35 | 1.0 s | 高单发，换弹快 |
| 狙击枪 | SR-X1 | 5 发 | 85 | 2.5 s | 极高单发，射速慢 |

### AI 敌人（单人模式）

AI 作为服务端机器人玩家接入，与真人共用同一套权威战斗逻辑。战术补给模式下，Bot 会在低血、低甲、缺弹时倾向前往拾取补给。三档难度：

| 难度 | 反应 | 瞄准 | 走位 |
|---|---|---|---|
| 简单 | 慢 | 误差大 | 保守 |
| 普通 | 适中 | 适中 | 适中 |
| 困难 | 快 | 精准 | 激进绕后侧移 |

## 操作

### 移动端

| 控件 | 位置 | 作用 |
|---|---|---|
| 虚拟摇杆 | 左下角 | 前后左右移动 |
| 滑动 | 屏幕右侧 | 转动视角 / 瞄准 |
| 射击 | 右下角大按钮 | 开火 |
| 换弹 | 射击上方 | 装填 |
| 跳跃 | 右侧 | 跳跃越障 |
| 武器槽 | HUD 底部 | 切换 AR-30 / PX-7 / SR-X1 |

> 移动端按钮位置支持**自定义拖拽布局**，见下方「设置面板」说明。

### 桌面端

`WASD` 移动，鼠标锁定后转向，左键射击，`R` 换弹，`空格` 跳，`1` / `2` / `3` 切枪（可在设置中自定义按键）。

## 设置面板

游戏内可调整（写入 localStorage，即时生效）：

| 选项 | 说明 |
|---|---|
| 灵敏度 | 视角转动速度 |
| 音量 | 全局音效音量 |
| 按钮尺寸 | 移动端触控按钮大小 |
| 画质 | 渲染分辨率（高/中/低） |
| 按键布局 | 桌面端自定义键位，持久化保存 |
| 按钮位置 / 编辑布局 | 移动端：进入布局编辑态，拖拽调整各按钮位置；支持保存、取消、重置默认 |

## 运行方式

需要 Node.js ≥ 16。

```bash
cd arena-fps
npm install      # 安装 express + socket.io
npm start        # 启动服务端（默认端口 3000，可用 PORT 环境变量覆盖）
```

### 本机测试
浏览器打开两个标签页访问 `http://localhost:3000`。

### 手机对战（同一局域网）
1. 确保电脑和手机连同一 Wi-Fi。
2. 查看电脑局域网 IP：macOS 用 `ipconfig getifaddr en0`，Windows 用 `ipconfig`。
3. 手机浏览器访问 `http://<电脑IP>:3000`，一台创建、一台输房间号加入。

> 进入对战时会尝试请求全屏与横屏锁定，竖屏同样可玩。

## 联机与反作弊设计

- **移动/朝向**：客户端预测上报，服务端转发（低延迟）。
- **战斗裁决（服务端权威）**：射击只发方向，服务端射线判定命中/遮挡，自行扣血/记分/击杀/复活。
- **武器裁决**：切枪/弹药/换弹/伤害全部由服务端管理，客户端无法篡改。
- **补给裁决**：拾取事件、护甲值、加速状态均由服务端裁决并广播。
- **限制**：弹药校验、射速上限、房间上限 2 人、加时先击杀即胜。
- **异常处理**：断线 10 秒宽限可重连；超时则对手收到「对手已离开」并结算。

## 性能

- 移动端优先：低面数 low-poly GLB、本地 Three.js、受控粒子数、阴影贴图 1024、像素比上限 2。
- 帧率目标 30 FPS +，低性能设备可流畅运行。
